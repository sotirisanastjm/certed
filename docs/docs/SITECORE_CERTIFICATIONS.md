# Sitecore Certifications Reference

Complete list of all Sitecore professional certifications organized by category.

---

## 1. Platform DXP (Digital Experience Platform)

### Sitecore 10 .NET Developer Certification
- **Target Audience**: Developers with .NET experience
- **Exam Format**: Multiple-choice, proctored via Kryterion Webassessor
- **Passing Score**: 80%
- **Exam Type**: Closed Book
- **Support Portal Access**: Yes
- **Key Competencies**:
  - Sitecore structure & platform
  - System Administration for Developers
  - Configuration Management
  - Item, Template, and Data Management
  - Renderings and Layout
  - Content Search
  - Sitecore API
  - Security
  - Content Serialization

### Sitecore 10 System Administrator Certification
- **Target Audience**: IT administrators managing Sitecore environments
- **Exam Format**: Multiple-choice, proctored via Kryterion Webassessor
- **Passing Score**: 80%
- **Exam Type**: Closed Book
- **Support Portal Access**: Yes
- **Key Competencies**:
  - Control Panel navigation and settings
  - Workflow and publishing
  - User and role management
  - Content Author support
  - Sitecore architecture and containers
  - Installation, updates, and upgrades
  - Performance and health monitoring
  - Sitecore Support Services

## 5. XM Cloud (Modern Headless CMS)

### Sitecore XM Cloud Developer Certification
- **Target Audience**: Developers building headless solutions
- **Exam Format**: 50 multiple-choice questions, 100 minutes
- **Passing Score**: 80%
- **Exam Type**: Closed Book
- **Support Portal Access**: No
- **Key Competencies**:
  - XM Cloud architecture (SaaS, headless)
  - Deployment and CI/CD (XM Cloud Deploy)
  - Headless rendering with JSS (Next.js)
  - Content serialization (Sitecore CLI)
  - GraphQL and REST APIs
  - Experience Edge (global CDN)
  - Security and access management
  - Data modeling and templates

### SitecoreAI CMS for Developers Certification
- **Target Audience**: Developers working with SitecoreAI platform
- **Exam Format**: Multiple-choice, proctored exam
- **Passing Score**: 80%
- **Exam Type**: Closed Book

- **Key Competencies**:

## Exam Information

- **Proctoring**: Most exams are proctored through Kryterion Webassessor

- **Validity**: Check Sitecore Learning for current recertification requirements
- **Preparation**: eLearning courses available at [Sitecore Learning](https://learning.sitecore.com)
- **Registration**: Purchase exams through Kryterion Webassessor
