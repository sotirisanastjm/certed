# Sitecore Certifications Reference Guide

## Overview

This document provides a comprehensive reference of all Sitecore certifications, their exam topics, official study resources, and recommended learning materials for the CertCore study application.

---

## Certification Categories

### 1. Platform DXP (Sitecore Experience Platform & Manager)

#### Sitecore 10 .NET Developer Certification

**Exam Details:**
- Questions: 60 multiple-choice
- Duration: 120 minutes
- Passing Score: 80%
- Format: Closed book, proctored via Kryterion Webassessor
- Grants: Sitecore Support Portal access

**Competency Breakdown:**

| Competency | Weight | Questions |
|------------|--------|-----------|
| Sitecore Structure & Platform | 12% | ~7 |
| Security & User Management | 16% | ~10 |
| Item Management | 20% | ~12 |
| Layout & Placeholders | 10% | ~6 |
| Components, Controls, & Renderings | 26% | ~16 |
| Sitecore Content Serialization | 10% | ~6 |
| Containers | 6% | ~3 |

**Key Topics:**
- Patching, multisite setups, domain bindings
- User roles, role precedence, unlocking users
- Templates, standard values, workflows
- Dynamic placeholders, versioned layouts
- MVC rendering, rendering parameters, view models
- Sitecore.json and serialization modules
- Container architecture and development

**Official Resources:**
- [Sitecore Learning - Developer's Fundamentals 9.3](https://learning.sitecore.com)
- [Sitecore Learning - Developer's Fundamentals 10.0](https://learning.sitecore.com)
- [Sitecore Documentation](https://doc.sitecore.com/xp)

**Recommended Study Materials:**
- Developer eLearning Certification Bundle ($880 USD)
- Sitecore 10 Developer Certification Study Guide
- Sitecore XP Documentation

## Study Tips

1. **Complete Official eLearning:** The eLearning courses cover 80-90% of exam content
2. **Use Practice Exams:** Familiarize yourself with question format and timing
3. **Review Documentation:** Official docs are the primary source for exam questions
4. **Hands-on Experience:** Practical experience reinforces theoretical knowledge
5. **Time Management:** Practice with timed quizzes to develop pacing skills
6. **Study Guides:** Download and follow official study guides
7. **Community Resources:** Leverage blogs and community posts from certified professionals
